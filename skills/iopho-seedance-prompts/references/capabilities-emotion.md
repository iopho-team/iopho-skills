# 情绪演绎更好

Seedance 2.0 在角色情绪表达上有了质的飞跃，能够精准演绎崩溃、温馨、喜剧、紧张、反转等复杂情绪。

## 核心能力

- 情绪渐变（从平静到崩溃）
- 多角色情绪互动
- 喜剧反转
- 紧张悬疑氛围
- 温馨感人场景

## Case 示例

### Case 1: 情绪崩溃

**素材**: @图片 1-2（女子形象）、@视频 1（情绪参考视频）
**prompt**:
```
@图片 1 的女子走到镜子前，看着镜子里面的自己，姿势参考@图片 2，
沉思了一会突然开始崩溃大叫，抓镜子的动作崩溃大叫的情绪和表情完全参考@视频 1。
```

**要点**: `崩溃大叫的情绪和表情完全参考@视频 1` — 情绪强度直接参考视频。

### Case 2: 油烟机广告 — 情绪对比

**素材**: @图片 1-3（女人做饭+男人做饭+油烟机）
**prompt**:
```
这是一个油烟机广告。@图片 1 作为首帧画面，女人在优雅的做饭，没有烟雾，
镜头快速向右边摇动，拍摄@图片 2 男人满头大汗面红耳赤在做饭，浓烟滚滚，
镜头向左边摇动推进拍摄@图片 1 桌面上的一个油烟机，油烟机参考@图片 3，
油烟机在疯狂抽烟。
```

**要点**: 通过两个角色的**情绪对比**（优雅 vs 满头大汗）突出产品价值。

### Case 3: 搞笑变身

**素材**: @图片 1-4（人物+表情+熊形象）
**prompt**:
```
@图片 1 作为画面的首帧图，镜头旋转推近，人物突然抬头，
人物面部长相参考@图片 2，开始大声咆哮，激动带有一些喜剧色彩，
参考@图片 3 的表情神态。然后人物身体变身成为一只熊，参考@图片 4。
```

**要点**: 情绪递进（平静→抬头→咆哮→变身）+ 喜剧色彩 + 多图分别参考不同维度。

### Case 4: 吃泡面的抽象行为

**素材**: @图片 1（人物）、@视频 1（参考动作/表情）
**prompt**:
```
@图片 1 的人物参考@视频 1 中的动作和表情变化，展示吃泡面的抽象行为
```

**要点**: 极简 prompt，`抽象行为` 让模型发挥创意，情绪表达由参考视频决定。

## 情绪关键词速查

| 情绪类型 | 描述关键词 |
|---------|----------|
| **崩溃** | 崩溃大叫、情绪失控、泪流满面、歇斯底里 |
| **温馨** | 温馨、感动、拥抱、微笑、泪目 |
| **喜剧** | 搞笑、诙谐幽默、夸张表情、喜剧色彩 |
| **紧张** | 紧张、焦虑、心虚、窘迫、严阵以待 |
| **愤怒** | 咬牙切齿、怒吼、拍桌子、面色铁青 |
| **惊恐** | 惊恐、瞪大眼睛、后退、尖叫 |
| **满足** | 满足、一脸享受、心满意足、陶醉 |
| **冷酷** | 冷静、冰冷、面无表情、气场全开 |
| **反转** | 颠覆、突然、出人意料、急转直下 |

## Prompt 写作要点

1. **情绪要具体**：不说"表现情绪"，说"崩溃大叫""满头大汗面红耳赤"
2. **情绪渐变比突变好**：`沉思了一会 → 突然开始崩溃大叫`
3. **对比制造冲击**：两个角色/场景的情绪对比（优雅 vs 狼狈）
4. **参考视频情绪**：`情绪和表情完全参考@视频 1`
5. **喜剧靠细节**：具体的搞笑动作比写"搞笑"更有效
6. **身体语言很重要**：描述肢体动作来表达情绪（握拳、后退、歪头）
