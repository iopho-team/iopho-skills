# Prompt 专业词库

Seedance 2.0 提示词中常用的专业术语和修饰词汇。中文优先，因为 Seedance 中文理解效果最佳。

## 5. 时间分段控制语法

```
[时间段]：[画面描述]

例:
0-3 秒：固定镜头，[场景描述]
3-6 秒：镜头推近，[动作描述]
6-10 秒：镜头拉远，[全景展示]
10-15 秒：[收尾/文案展示]
```

### 节奏关键词

| 节奏 | 关键词 |
|------|--------|
| 快 | 快速闪切、急速、节奏紧凑 |
| 慢 | 缓慢推进、匀速、节奏舒缓 |
| 变速 | 先慢后快、骤然加速、戛然而止 |
| 卡点 | 根据音乐节奏卡点、画面关键帧同步 |

## 使用建议

1. **中文优先**：Seedance 中文理解最佳，直接用中文描述
2. **组合使用**：`[运镜] + [景别] + [主体] + [动作] + [氛围] + [风格]`
3. **不需要全部堆叠**：选择最关键的 3-5 个修饰词即可
4. **具体优于抽象**：`满头大汗面红耳赤` 优于 `很热的样子`
5. **动词优于形容词**：`镜头急速穿梭` 优于 `很快的镜头`
