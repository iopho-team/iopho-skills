# 音色更准，声音更真

Seedance 2.0 的声音生成能力大幅提升，支持对话配音、音效生成、音色参考、BGM 节奏匹配。

## 核心能力

- 自然对话配音（多角色、多语言）
- 环境音效自动生成
- 参考视频/音频的音色复刻
- BGM 节奏与画面同步
- 方言/特殊口音支持

## Case 示例

### Case 1: 鱼眼镜头 + 音色参考

**素材**: @视频 1-3（鱼眼运镜 + 马匹视频 + 音效参考）
**prompt**:
```
固定镜头，中央鱼眼镜头透过圆形孔洞向下窥视。参考视频 1 的鱼眼镜头，
让@视频 2 中的马看向鱼眼镜头，参考@视频 1 中的说话动作，
背景 BGM 参考@视频 3 中的音效。
```

**要点**: 三个视频分别参考不同维度 — 运镜(@视频 1)、主体(@视频 2)、音效(@视频 3)。

### Case 2: 地产纪录片 — 旁白音色

**素材**: @图片 1-3（写字楼宣传照）、@视频 1（旁白音色参考）
**prompt**:
```
根据提供的写字楼宣传照，生成一段 15 秒电影级写实风格的地产纪录片。
采用 2.35:1 宽银幕，24fps，细腻的画面风格，其中旁白的音色参考@视频 1，
拍摄"写字楼的生态"，呈现楼内不同企业的运作，
结合旁白阐述写字楼如何成为一个充满活力的商业生态系统。
```

**要点**: `旁白的音色参考@视频 1` — 精确指定音色来源。

### Case 3: 方言对话 — 猴子奶茶店

**素材**: @图片 1（猴子形象）、@图片 2（比熊服务员形象）
**prompt**:
```
@图片 1 的猴子走向奶茶店柜台，镜头跟随在他身后。一位@图片 2 的比熊服务员正在吧台处
擦拭制作工具，猴子向服务员用四川口音点单："幺妹儿，霸王别姬有得没得？"
切镜，特写。服务员放下手里的活，怪异地看了老头一眼后回答："没得，美式要不要得嘛"
切镜，镜头给到猴子。他挠了挠头念念有词："没事……？我有事！孙儿叫我来买个奶茶，
就叫个撒子霸王别姬嘛"
```

**要点**: 对话直接写在 prompt 中，包含方言（四川口音）。模型自动配音。

### Case 4: 多角色多语言对话

**prompt**:
```
画面中间戴帽子的女孩温柔地唱着说"I'm so proud of my family!"，之后转身拥抱中间的
黑人女孩。黑人女孩感动地回应"My sweetie, you're the heart of our family"，
回抱她。左侧的黄衣服男孩开心地说"Folks, let's dance together to celebrate!"
最右侧的女孩紧接着回复："I'll bring the music!"，背景拉美音乐起，
左侧穿橙色裙的女性（朱丽叶塔）笑着点头，右侧扎辫女性（路易莎）握紧拳头挥动手臂。
人群中有人开始踏起步子，孩子们跟着节奏拍手，整个家族即将围成圈，
伴着欢快的音乐，裙摆飞扬，在五彩的街道上尽情舞动，传递着喜悦与温暖。
```

**要点**: 多角色英文对话 + 音乐 + 舞蹈动作，全部在一个 prompt 中。

### Case 5: 军事/战术对话

**prompt**:
```
固定镜头。站着的壮汉（队长）握拳挥臂用西班牙语说着："三分钟后突袭！"，
金发队员站在检查枪械，绿发队员握紧战术手电。黑人队员搭肩问同伴用西班牙语说：
"侧翼包抄？"队长点头用西班牙语说："老规矩，活口留审讯。"
全员肃然，装备碰撞声中完成战术手势，默契起身，大家都严阵以待。
```

### Case 6: 起床场景 — 音色参考

**素材**: @图片 1-2（角色形象）、@视频 1（语气音色参考）
**prompt**:
```
0-3 秒：开头闹钟响起来，画面朦胧中出现画面 1；
3-10 秒：快速摇镜头，转向对面特写男人面部，男人无奈的叫女生起床，
语气和音色参考@视频 1；
10-12 秒：女生撇着嘴躲进被子里面；
12-15 秒：切换到男主全身，他叹着气说："真拿你没办法！"
```

**要点**: `语气和音色参考@视频 1` — 不只是"声音"，还包括"语气"。

## 音频相关关键词

| 维度 | 关键词 |
|------|--------|
| 语气 | 温柔、冷静、激动、无奈、调侃、严肃 |
| 音色 | 参考@视频/音频、低沉、清亮、沙哑 |
| 方言 | 四川口音、东北腔、粤语、台湾腔 |
| 语言 | 普通话、英语、西班牙语、日语、韩语 |
| 环境音 | 走路声、人群声、汽车声、风声、雨声 |
| 音效 | 咔嗒声、碰撞声、爆炸声、翻书声 |
| BGM | 背景音乐参考@视频、轻快、紧张、恢宏大气 |

## Prompt 写作要点

1. **对话直接写**：在 prompt 中直接写角色台词，包含标点和语气
2. **音色参考**：`语气和音色参考@视频 1` 或 `参考@音频 1 的音色`
3. **方言要标注**：`用四川口音`、`东北腔说`
4. **环境音描述**：直接在场景描述中写声音（"传来脚步声""背景音效为走路声"）
5. **BGM 独立指定**：`背景 BGM 参考@视频 X 中的音效`
6. **多语言混合**：不同角色可以说不同语言
