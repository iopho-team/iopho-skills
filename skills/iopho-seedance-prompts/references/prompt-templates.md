# Prompt 模板库（按场景分类）

从官方手册 40+ 个 Case 中提炼的可复用 prompt 模板。

## 1. 电商产品广告

### 模板 A: 产品特写 + 功能展示

```
0-2 秒画面：[产品多角度快速闪切]，特写[材质/质感]与"[品牌名]"品牌字样。
3-6 秒画面：特写[核心卖点动作]，展示[质感/便捷性]。
7-12 秒画面：快速切换[使用场景1]；[使用场景2]；[使用场景3]；[使用场景4]。
13-15 秒画面：[产品陈列]，品牌名展示。
```

### 模板 B: 广告词 + 产品画面

```
参考视频的广告创意，用提供的[产品]图片，并参考[材质/元素]图片，
搭配以下广告词"[广告文案]"，生成新的[产品类型]广告视频。
```

### 模板 C: 油烟机式对比广告

```
@图片 1 作为首帧画面，[正面场景描述]，镜头快速向[方向]摇动，
拍摄@图片 2 [反面场景描述]，镜头[运动]拍摄[产品]，[产品功能展示]。
```

## 2. 剧情短片

### 模板 A: 颠覆剧情

```
颠覆@视频 1 里的剧情。
0–3 秒画面：[原始场景铺垫，氛围营造]
3–6 秒画面：[对话/情绪铺垫]
6–9 秒画面：[剧情反转点]
9–12 秒画面：[角色反应]
13–15 秒画面：[结局 + 字幕/文案]
```

### 模板 B: 追逐/动作戏

```
镜头跟随[角色]快速[动作]，后面[追逐者描述]。镜头转为侧面跟拍，
[碰撞/互动细节]，[环境音效描述]。
```

### 模板 C: 温馨日常

```
[角色]@图片 1 [日常场景描述]，[动作细节1]，[动作细节2]，
[情绪转折]，[温馨结尾]，全程自然对话
```

## 3. MV / 音乐视频

### 模板 A: 音乐卡点换装

```
[角色描述]在不停的换装，服装参考@图片 1@图片 2 的样式，
手中提着@图片 3 的[道具]，视频节奏参考@视频
```

### 模板 B: 风光卡点蒙太奇

```
@图片 1@图片 2@图片 3@图片 4@图片 5@图片 6 中的图片
根据@视频中的画面关键帧的位置和整体节奏进行卡点，
画面中的[主体]更有动感，整体画面风格更[氛围]，
可根据音乐及画面需求自行改变参考图的景别，及补充画面的光影变化
```

### 模板 C: 情绪向 MV

```
生成一个 15 秒的 MV 视频。关键词：[视觉关键词列表]
[镜头运动描述]，[主体与环境]，[光影氛围]，[画面风格]，[胶片/质感修饰]。
```

## 4. 一镜到底 / 长镜头

### 模板 A: 多场景穿梭

```
@图片 1@图片 2@图片 3@图片 4@图片 5，一镜到底的[类型]镜头，
从[起点]跟随[主体][动作]，穿过[场景1]，进入[场景2]，最终[结局]。
```

### 模板 B: 谍战/悬疑一镜到底

```
[类型]风格，@图片 1 作为首帧画面。镜头正面跟拍[角色描述]，
镜头全景跟随，[中间事件]，走到[转折点]，[新角色出现]，
面具女孩形象参考@图片 3，只参考形象。全程不要切镜头，一镜到底。
```

### 模板 C: 主观视角

```
@图片 1@图片 2...@图片 N，主观视角一镜到底的[场景类型]的镜头，
[速度/节奏变化描述]。
```

## 5. 科普 / 教育

### 模板 A: 图解演绎

```
用科普风格和音色，将图片 1 中的内容演绎出来，内容包括[知识点概述]
```

### 模板 B: 分镜脚本生成

```
参考@图片 1 的[类型]的分镜头脚本，参考@图片 1 的分镜、景别、运镜、
画面和文案，创作一段 [时长] 的关于"[主题]"的[风格]片头
```

## 6. 搞笑 / 创意

### 模板 A: 动物拟人对话

```
在"[场景名]"里的一段[对话类型]，要求情感丰沛，符合[表演风格]：
[角色A]（[角色特征]）："[台词]"
[角色B]（[角色特征]）："[台词]"
```

### 模板 B: 文化创意混搭

```
[传统元素]的[具体桥段]响起，[角色A描述]指着[角色B描述]，
[动作和台词描述]，[剧情发展]，[转折/意外事件]
```

### 模板 C: 抽象/脑洞

```
延长 [时长] 视频，参考@图片 1、@图片 2 的[荒诞组合]，补充一段脑洞广告
画面 1：[荒诞场景1]
画面 2：[特技/夸张动作]
画面 3：[收尾 + 广告词]
```

## 7. 视频延长

### 模板 A: 叙事延长

```
向前延长 [时长]，[场景铺垫描述]，[细节动作]，[情绪/氛围]。
```

### 模板 B: 广告延长

```
将视频延长 [时长]。
[时间段1]：[画面描述]
[时间段2]：[画面描述]
[时间段3]：[文案/logo展示]
```

## 使用说明

1. 选择最匹配的模板类别
2. 将 `[占位符]` 替换为具体内容
3. 根据素材情况调整 @引用
4. 时长超过 10s 建议加时间分段
5. 可以组合多个模板（如"产品广告 + 音乐卡点"）
