# 基础能力：更稳、更顺、更像真的

Seedance 2.0 在基础层面显著增强：**物理规律更合理**、**动作表现更自然流畅**、**指令理解更精准**、**风格保持更稳定**。不仅能稳定完成复杂动作、连续运动等高难度生成任务，也让整体视频效果更真实、更顺滑。

## 核心提升

- 物理规律合理（重力、碰撞、流体）
- 动作连贯自然（走路、跑步、格斗、舞蹈）
- 指令精准响应（描述什么生成什么）
- 风格全程一致（色调、画风不跳变）

## Case 示例

### Case 1: 超强真实感 — 日常动作

**prompt**:
```
女孩在优雅的晾衣服，晾完接着在桶里拿出另一件，用力抖一抖衣服。
```

**要点**: 纯文本描述，模型自行理解物理动作（抖衣服的布料飘动、重力感）。

### Case 2: 电影级复杂叙事

**prompt**:
```
画里面的人物心虚的表情，眼睛左右看了看探出画框，快速的将手伸出画框拿起可乐喝了一口，
然后露出一脸满足的表情。这时传来脚步声，画中的人物赶紧将可乐放回原位。
此时一位西部牛仔拿起杯子里的可乐走了，最后镜头前推画面慢慢变得纯黑背景只有顶光照耀的
罐装可乐，画面最下方出现艺术感字幕和旁白："宜口可乐，不可不尝！"
```

**要点**: 长叙事 prompt，包含多角色、情绪变化、镜头切换、音效、字幕。Seedance 2.0 能一次生成完整故事。

### Case 3: 复杂场景 — 追逐戏

**prompt**:
```
镜头跟随黑衣男子快速逃亡，后面一群人在追。镜头转为侧面跟拍，
人物惊慌撞倒路边的水果摊爬起来继续逃，人群慌乱的声音。
```

**要点**: 多角色互动 + 环境破坏 + 镜头跟随 + 音效描述。物理碰撞（水果摊倒塌）表现自然。

### Case 4: 伦敦街头 — 风格化场景

**prompt**:
```
镜头小幅度拉远（露出街头全景）并跟随女主移动，风吹拂着女主的裙摆，
女主走在19世纪的伦敦大街上；女主走着走着右边街道驶来一辆蒸汽机车，
快速驶过女主旁，风将女主的裙摆吹起，女主一脸震惊的赶忙用双手向下挡住裙摆；
背景音效为走路声、人群声、汽车声等等
```

**要点**: 时代场景构建 + 物理细节（风吹裙摆、蒸汽机车气流）+ 角色情绪反应 + 多层音效。

## Prompt 写作要点

1. **动作要具体**：不说"做动作"，说"用力抖一抖衣服""赶紧将可乐放回原位"
2. **物理细节自然涌现**：描述场景，模型自动补充物理效果（布料飘动、碰撞）
3. **叙事可以长**：Seedance 2.0 理解长文本叙事，不需要刻意缩短
4. **音效直接写**：在 prompt 中直接描述环境音，模型会生成对应声音
5. **情绪要明确**：写清楚角色的表情和情绪状态（心虚、满足、震惊）
