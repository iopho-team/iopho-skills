# 高难度/可控的运镜和动作精准复刻

以前想让模型模仿电影里的走位、运镜或者复杂动作，要么写一堆细节提示词，要么干脆做不到。而现在，只需要上传一段参考视频，就可以了。

## 核心能力

- 参考视频复刻镜头语言（推拉摇移环绕）
- 复杂运动轨迹（追踪、绕行、穿梭）
- 一镜到底（长镜头无剪切）
- 希区柯克变焦等专业手法
- 多场景穿梭转场

## Case 示例

### Case 1: 电梯追踪运镜

**素材**: @图片 1-3（人物+场景）、@视频 1（参考运镜）
**prompt**:
```
参考@图片 1 的男人形象，他在@图片 2 的电梯中，完全参考@视频 1 的所有运镜效果
还有主角的面部表情，主角在惊恐时希区柯克变焦，然后几个环绕镜头展示电梯内视角，
电梯门打开，跟随镜头走出电梯，电梯外场景参考@图片 3，男人环顾四周，
参考@视频 1 用机械臂多角度跟随人物的视线
```

**要点**: `完全参考@视频 1 的所有运镜效果` + 具体运镜描述（希区柯克变焦、环绕、机械臂跟随）。

### Case 2: 走廊追逐 — 多角度切换

**素材**: @图片 1-5（人物+场景）、@视频 1（参考运镜）
**prompt**:
```
参考@图片 1 的男人形象，他在@图片 2 的走廊中，完全参考@视频 1 的所有运镜效果，
还有主角的面部表情，镜头跟随主角在@图片 2 拐角奔跑，然后在@图片 3 的长廊里，
镜头从背面的跟随视角，通过低视角环绕到主角正面；镜头再右摇 90 度拍摄@图片 4 的分叉路口，
急停后右摇 180 度，怼脸拍摄主角正面；主角气喘吁吁，镜头跟随主角的视角环顾四周，
参考@视频 1 里急速的左右环绕运镜展示场景，后拉到@图片 5 的场景，继续跟拍主角奔跑的侧面视角
```

**要点**: 极度详细的运镜指令 — 角度（低视角、怼脸）+ 方向（左环绕、右摇 90/180 度）+ 运动（跟随、急停、后拉）。多个 @图片 指定不同场景。

### Case 3: 产品展示环绕

**素材**: @图片 1（平板电脑）、@视频 1（聚焦旋转参考）
**prompt**:
```
@图片 1 的平板电脑作为主体，运镜参考@视频 1，推近到屏幕的特写，
镜头旋转后平板反转展示全貌，屏幕中的数据流一直在变化，
周围的环境逐渐变成科幻风格的数据空间
```

**要点**: 产品环绕运镜 + 环境风格变化（写实 → 科幻）。

### Case 4: 舞蹈运镜复刻

**素材**: @图片 1（女星形象）、@视频 1（推拉舞蹈运镜参考）
**prompt**:
```
@图片 1 的女星作为主体，参考@视频 1 的运镜方式进行有节奏的推拉摇移，
女星的动作也参考@视频 1 中女子的舞蹈动作，在舞台上活力十足地表演
```

**要点**: 同时参考运镜 + 动作，两个维度从同一个视频提取。

### Case 5: 一镜到底 — 追踪长镜头

**素材**: @图片 1-5（多个场景关键帧）
**prompt**:
```
@图片 1@图片 2@图片 3@图片 4@图片 5，一镜到底的追踪镜头，
从街头跟随跑步者上楼梯，穿过走廊，进入屋顶，最终俯瞰城市。
```

**要点**: 多图作为场景关键帧序列，`一镜到底` 指令让模型生成无剪切长镜头。

### Case 6: 谍战风 — 一镜到底

**素材**: @图片 1-4（角色+场景）
**prompt**:
```
谍战片风格，@图片 1 作为首帧画面。镜头正面跟拍穿着红风衣的女特工向前走，
镜头全景跟随，不断有路人遮挡红衣女子，走到一个拐角处，参考@图片 2 的拐角建筑，
固定镜头红衣女子离开画面，走在拐角处消失，一个戴面具的女孩在拐角处躲着恶狠狠的盯着她，
面具女孩形象参考@图片 3，只参考形象，女孩站在拐角处。镜头往前摇向红衣女特工，
她走进一座豪宅消失不见了，豪宅参考@图片 4。全程不要切镜头，一镜到底。
```

**要点**: `全程不要切镜头，一镜到底` 是关键指令。每个 @图片 精确指定角色/场景。

### Case 7: 木屋温馨 — 多动物场景

**素材**: @图片 1-4（外景+动物+道具）
**prompt**:
```
根据@图片 1 外景的镜头，第一人称主观视角快推镜头到木屋内的环境场景近景，
一只小鹿@图片 2 和一只羊@图片 3 在围炉旁喝茶聊天，
镜头推进特写茶杯的样式参考@图片 4。
```

### Case 8: 过山车主观视角

**素材**: @图片 1-5（多个场景图）
**prompt**:
```
@图片 1@图片 2@图片 3@图片 4@图片 5，主观视角一镜到底的惊险过山车的镜头，
过山车的速度越来越快。
```

**要点**: 多图 + 主观视角 + 一镜到底，极简 prompt 但效果震撼。

## 运镜关键词速查

| 运镜类型 | 关键词 |
|---------|--------|
| 推近 | 推近、push in、zoom in、前推 |
| 拉远 | 拉远、pull out、zoom out、后拉 |
| 环绕 | 环绕镜头、orbit、围绕主体旋转 |
| 追踪 | 跟随镜头、跟拍、tracking shot |
| 摇移 | 左摇、右摇、pan left/right |
| 升降 | 升镜头、降镜头、crane up/down |
| 一镜到底 | 一镜到底、长镜头、全程不切镜头 |
| 希区柯克 | 希区柯克变焦、dolly zoom |
| 主观视角 | 第一人称视角、POV、主观镜头 |
| 固定 | 固定镜头、static shot |
| 俯拍 | 俯拍视角、鸟瞰、overhead |
| 仰拍 | 仰拍、low angle、低角度 |
| 手持 | 手持镜头、handheld |

## Prompt 写作要点

1. **运镜参考 + 内容替换**：`参考@视频 1 的运镜` + 自己描述内容
2. **角度要精确**：写清楚"右摇 90 度""低视角环绕到正面"
3. **一镜到底要强调**：开头或结尾写 `一镜到底` 或 `全程不要切镜头`
4. **多图 = 多场景**：每张图代表一个场景节点，镜头按顺序穿过
5. **速度可控**：`缓慢推近`、`急速环绕`、`匀速跟随`
